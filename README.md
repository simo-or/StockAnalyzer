# StockAnalyzer

StockAnalyzer is a Python application that analyzes publicly traded companies using the Yahoo Finance API.

## Features

- Retrieves company information
- Displays current stock price
- Shows daily high and low prices
- Displays market capitalization
- Calculates P/E ratio
- Displays dividend yield
- Calculates beta (risk indicator)
- Generates a simple investment score
- Provides a basic recommendation based on financial metrics

## Technologies

- Python
- yfinance
- Git
- GitHub

## How to Run

1. Install the required library:

```bash
pip install yfinance
```

2. Run the application:

```bash
python main.py
```

## Example Output

The program analyzes a stock and displays:

- Company information
- Financial indicators
- Risk level
- Investment score
- Final recommendation
